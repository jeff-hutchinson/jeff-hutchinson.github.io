# Mission: Laurieville Farms - Astro Build Phase

## Objective
Build a high-performance static site using Astro, optimized for local SEO and future migration to Wix.

## Architecture
- **Framework:** Astro (Static Site Generation).
- **Navigation:** Single "Products" top-level menu with sub-categories (Honey, Beeswax, Supplies).
- **Data Strategy:** Store product descriptions and flyer text in `src/content/products/` as Markdown files.

## Design Requirements (Inspired by Bee Lovely Botanicals)
- **Responsive:** Mobile-first stacking for all product grids.
- **Components:** Modular Header and Footer for "Single Source of Truth."
- **SEO:** Automated Schema.org generation for each product page.
